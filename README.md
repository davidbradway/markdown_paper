## Compiling the manuscript

I'm using [`Pandoc`](http://johnmacfarlane.net/pandoc/) as the document generator. Once installed, run:
```
git clone git@github.com:davidbradway/markdown_paper.git 
cd markdown_paper
./knit.sh
```
